Projeto Final – Website em HTML e CSS
Objetivo:
Desenvolver um website original, em HTML e CSS, sobre um tema à tua escolha.
O objetivo é aplicar os conteúdos aprendidos e acrescentar novos conhecimentos através de
pesquisa autónoma.
Tema:
O tema da página é à escolha do aluno.
Exemplos: página institucional, site de apresentação de negócio fictício, landing page de evento,
ou portefólio. O tema deve permitir aplicar os elementos solicitados.
Requisitos obrigatórios
O projeto deve incluir:
Mínimo 2 páginas HTML
Navegação entre páginas (navbar ou menu)
Uso adequado de tags semânticas (header, nav, main, footer, etc.)
Conteúdo:
• Título e subtítulos
• Texto organizado
• Pelo menos 2 imagens
• Links internos e externos
CSS
• Ficheiro CSS externo
• Navbar funcional
• Seletores claros, classes bem nomeadas.
• Esquema de cores e tipografia consistente.
• Flex ou grid; margens, paddings e personalização de links e menus.
• Sem frameworks.
Pesquisa adicional (obrigatório)
• Layout responsivo
Deves incluir pelo menos 2 elementos que não foram dados diretamente nas aulas.
Exemplos:
• Google Fonts
• Animações simples
• Transições
• Ícones
• Gradientes
• Efeitos visuais
Entrega
Todos os ficheiros organizados numa pasta:
• HTML
• CSS
• Imagens
• O projeto deve funcionar corretamente ao abrir o index.html
Apresentação Oral
Cada aluno fará uma apresentação (5–8 minutos) onde deverá:
• Explicar a estrutura do projeto
• Mostrar o código
• Justificar escolhas feitas
• Explicar o que aprendeu de novo
• Demonstrar domínio do que fez
O aluno deve conseguir explicar o seu próprio código.
Caso não consiga justificar ou compreender o que desenvolveu, isso será considerado na
avaliação.
